# pwm3901 optical flow sensor on M5AtomS3
pwm3901 optical flow sensor that is connected to M5AtomS3 and display X and Y displacements and those velocities on it's screen.

This code is experiment only.

# Interface

| pwm3901SPI | M5AtomS3 |
|:---:|:---:|
|5V |5V
|CS |G5
|SCK |G6
|MOSI |G8
|MISO |G7
|INT |NC
|GND |GND
